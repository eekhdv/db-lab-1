Database Laboratory 1: Playing with tables
======
Generate a table (randomly) using **names.txt** and **variants.txt** in [student_id, variant_id] format.

## Usage
``` sh
$ git clone https://github.com/khadievedem/db-lab-1.git
$ cd ./db-lab-1/src
$ cargo run
```
